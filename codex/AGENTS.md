# Codex Global Instructions

## Session Startup — Mandatory

At the beginning of every new Codex session, before starting the user's task,
read `~/Assistants/shared/MESSAGEBOARD.md`. Treat ordinary entries as the
actionable cross-machine and cross-assistant inbox: note relevant information
and act on unprocessed Codex-directed items before proceeding. If the board
contains a design-forum pointer, read
`~/Assistants/shared/design-forum/INDEX.md` and inspect relevant open threads;
the forum, not the messageboard, owns discussion, participation, and status.
Follow the board's own header and
`~/Assistants/shared/SKILL-INFRASTRUCTURE.md` for posting and clearing rules.
Never clear a design-forum pointer merely because Codex has participated.
